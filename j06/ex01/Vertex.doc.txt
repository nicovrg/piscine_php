<- Vertex ----------------------------------------------------------------------
The Vertex class handles coordinates.

An instance can be contructed from 5 args:
int x, int y, int z, int w, class color 

The class provides the following methods :

get_data (data being either x, y, z, w or color);
get the value from the object

set_data (data being either x, y, z, w or color);
set the value from the object
